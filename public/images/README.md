# Esta es una carpeta para las imágenes del cliente

## Instrucciones:

1. Coloca aquí las fotos del cliente optimizadas
2. Nombres recomendados:
   - local-exterior.jpg
   - local-interior.jpg
   - producto-1.jpg
   - producto-2.jpg
   - etc.

3. Optimiza las imágenes ANTES de subirlas:
   - Usa https://tinypng.com (gratis)
   - Tamaño máximo: 1920x1080px
   - Peso máximo: 800KB por imagen

4. Actualiza el array `gallery` en config/client-config.ts con los nombres exactos
