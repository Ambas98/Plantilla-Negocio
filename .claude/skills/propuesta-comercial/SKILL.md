# Skill: Propuesta Comercial para Restaurantes

## Precios actuales
- Setup básico: $150,000 ARS
- Setup completo: $180,000 ARS
- Mantenimiento básico: $15,000 ARS/mes
- Mantenimiento completo: $20,000 ARS/mes

## Proceso
1. Verificar si existe research en docs/propuestas/
2. Leer docs/clientes.md para ver contexto
3. Personalizar con datos reales — nunca mandar propuesta genérica
4. Guardar en docs/propuestas/propuesta-[nombre-normalizado].md

## Personalización según situación del prospecto
- Muchas reseñas (200+): enfatizar "ya tenés clientes satisfechos, falta capturarlos online"
- Sin web: enfatizar pérdida de clientes que buscan online
- Web vieja/rota: enfatizar que la imagen actual los perjudica
- Buen IG: mencionar que el sitio web complementa las redes
