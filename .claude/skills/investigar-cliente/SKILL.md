# Skill: Investigar Cliente Potencial

## Fuentes a consultar
1. Búsqueda web: "[nombre] San Juan Argentina" y "[nombre] San Juan reseñas"
2. Google Maps: rating, cantidad de reseñas, categoría, horarios
3. Instagram/Facebook: cuenta activa, seguidores, última publicación
4. Sitio web actual: ¿existe? ¿funciona? ¿mobile-friendly? ¿tiene menú?
5. Apps de delivery: PedidosYa, Rappi

## Output
Guardar en docs/propuestas/research-[nombre-normalizado].md con:
- Datos básicos (rating, categoría, horarios)
- Presencia digital (web, redes, delivery)
- Lista de 3-5 oportunidades concretas
- Párrafo de evaluación de potencial como cliente
